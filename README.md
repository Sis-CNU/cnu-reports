# 1. CNU Reports.

## 1.1 Clonar e iniciar el proyecto.

1. `git clone git@github.com:Sis-CNU/CNU-reports.git`
2. `cd CNU-reports/`
3. `composer install`
4. `composer dump`
5. `php -S localhost:8000 -t public/`
